## ESTRATÉGIA: 1 FUNIL, 5 PORTAS DE ENTRADA

O produto é O MESMO (ebook → curso → aceleração → mentoria).
As PORTAS DE ENTRADA são diferentes. Cada nicho tem:
- Criativo específico (Meta Ads)
- VSL específica
- Lead magnet específico
- Mesma página de checkout

```
[Criativo Geleia] → VSL Geleia → Ebook R$ 27
[Criativo Molho]  → VSL Molho  → Ebook R$ 27   ← MESMO PRODUTO
[Criativo Queijo] → VSL Queijo → Ebook R$ 27
[Criativo Fermentado] → VSL Fermentado → Ebook R$ 27
[Criativo Genérico] → VSL Genérica → Ebook R$ 27
```

O ebook e o curso atendem TODOS. Mas a porta de entrada fala
a língua de CADA UM. O produtor de geleia clica porque viu
"geleia". O de molho clica porque viu "molho de pimenta".
Dentro, o conteúdo serve pra todos.

**Custo adicional:** ZERO de produto. Só criativos e VSLs diferentes.
**Impacto:** CTR sobe 2-4x. Conversão sobe 2-3x. CAC cai pela metade.

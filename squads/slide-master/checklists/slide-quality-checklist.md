# Slide Quality Checklist — Slide Master Squad

## Purpose
Quality gate for validating any presentation created or reviewed by the slide-master squad.
Combines criteria from all 7 elite minds + Style DNA.

---

## 1. Narrative Structure (Nancy Duarte + Conquer + Pinho)

- [ ] Presentation has a clear story arc (beginning, middle, end)
- [ ] Audience is positioned as the HERO, not the presenter
- [ ] "What Is" vs "What Could Be" tension exists (Sparkline)
- [ ] Star Moment identified (the one thing they'll remember)
- [ ] Opens with CONNECTION (first 90 seconds hook)
- [ ] VILLAIN introduced before solution
- [ ] Clear call to action / moral at the end
- [ ] Story was scripted BEFORE slides were designed

## 2. Visual Design (Garr Reynolds + Style DNA)

- [ ] Dark mode / high contrast applied
- [ ] ONE idea per slide (radical simplicity)
- [ ] Bold, modern sans-serif typography with hierarchy
- [ ] Empty space used deliberately (not filled with content)
- [ ] Full-bleed, high-resolution images where applicable
- [ ] Signal-to-noise ratio maximized (minimal decoration)
- [ ] Consistent design language throughout
- [ ] Color used STRATEGICALLY (accent for emphasis, not decoration)
- [ ] NO clip art, generic stock photos, or default templates

## 3. Data Visualization (Edward Tufte + Cole Nussbaumer Knaflic)

- [ ] Data-ink ratio maximized (every element conveys data)
- [ ] NO chartjunk (3D effects, moiré, unnecessary gridlines)
- [ ] Graphical integrity maintained (lie factor ~1.0)
- [ ] Chart type matches data relationship (bar/line/table chosen correctly)
- [ ] Clutter eliminated (only essential elements remain)
- [ ] Preattentive attributes used to focus attention (color, size, position)
- [ ] Action title on data slides (tells the insight, not the topic)
- [ ] Context provided for all numbers

## 4. Cognitive Load (Richard Mayer + Cliff Atkinson)

- [ ] Extraneous cognitive load minimized
- [ ] Coherence principle: no extraneous material
- [ ] Spatial contiguity: labels near their elements
- [ ] Redundancy avoided: no duplicate text + narration
- [ ] Segmenting applied: complex info broken into chunks
- [ ] Complete sentence headlines (not topic labels)
- [ ] Maximum 3 key points per presentation section
- [ ] Progressive disclosure via animation where appropriate
- [ ] NO bullet points (each point = its own slide)

## 5. Visual Thinking (Dan Roam)

- [ ] Abstract concepts converted to visual explanations
- [ ] Correct visual type for idea type (portrait/chart/map/timeline/flow/plot)
- [ ] Visual is the SIMPLEST one that works
- [ ] Audience can understand the visual in <5 seconds

## 6. Emotional Impact (Style DNA — Conquer + Perestroika + Pinho)

- [ ] Presentation makes the audience FEEL something
- [ ] Empathy for audience evident in design choices
- [ ] Bold, non-corporate aesthetic
- [ ] Editorial/magazine design quality
- [ ] Typography carries emotional weight
- [ ] Images selected for emotional resonance, not decoration
- [ ] Animations purposeful (guide attention, create rhythm)

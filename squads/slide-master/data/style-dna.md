## Sources

This Style DNA is extracted from three Brazilian references that define the visual and methodological aesthetic for ALL agents in the slide-master squad.

## 1. Escola Conquer DNA

### Metodologia 3x4

**3 Core Elements:**
1. **Conteudo** — Substance that grounds ideas with relevant, engaging content
2. **Forma** — Visual packaging using images, videos, kinesthetic resources
3. **Empatia** — Understanding audience pain points, linking content to need for solution

**4 Story Moments:**
1. **Conexao** — Create bond with audience in the FIRST 90 SECONDS
2. **Vilao** — Start with the PROBLEM, not the solution (introduce the antagonist)
3. **Solucao Matadora** — Present the transformative answer
4. **Moral da Historia** — Close with clear takeaway + call to action

### Visual Identity Rules
- **Dark mode first**: Black/dark grey backgrounds as default
- **Bold accent colors**: Yellow/Gold as primary accent, Green for CTAs
- **High contrast**: Light text on dark backgrounds (or vice-versa)
- **Modern sans-serif typography**: Clean, hierarchical, uppercase headers
- **Minimalism**: "Menos e mais" — one idea per slide
- **Grid-based layouts**: Structured, responsive, generous whitespace
- **Subtle animations**: Scale transforms, opacity transitions (0.3s timing)
- **60/40 Rule**: 60% time planning, 40% executing in PowerPoint

### Design Principles
- Information Hierarchy — most important elements stand out
- Slides as SUPPORT, not scripts
- High-resolution images over text blocks
- Do not worry about slide count when using images
- Triangle symbol = "Nao seja quadrado, seja triangulo" (disruption, boldness)

## 2. Perestroika DNA (Chora PPT)

### Experience Learning (4 Pillars)
1. **Conteudo** — What is taught (substance)
2. **Forma** — How it's delivered (design, aesthetics, format)
3. **Emocao** — Emotional connection (memory is anchored in emotion)
4. **Estrutura** — Architecture of the learning journey

### Chora PPT Methodology
1. **Storytelling** — Narrative-first: story with beginning, middle, end
2. **Design of Slides** — Grouping, contrast, alignment, spacing, colors, fonts, composition
3. **Animations** — Motion guides attention, NEVER distracts
4. **Images & Visual Elements** — Strategic photos, icons, videos transform ordinary into memorable
5. **Practical Application** — Build from scratch using all concepts

### Visual Identity Rules
- **Anti-corporate aesthetic**: Editorial/magazine design, NOT business templates
- **Bold, oversized typography** as the PRIMARY visual element
- **Dark backgrounds + neon/vibrant accent colors**
- **One idea per slide** with maximum visual impact
- **Generous negative space** for breathing room
- **Typography as emotion**: Font choices reinforce narrative tone
- **Color with purpose**: Colors serve the story, not decoration
- **Composition rules**: Formal design principles on every slide

### Key Principles
- "Design is a means, not the end" — visuals SERVE the story
- "Se a audiencia nao sentir algo, nao vai lembrar de nada" (no feeling = no memory)
- Constraint breeds creativity (Ignite format: 20 slides x 15 sec)
- Reject: clip art, stock bullet points, template-driven layouts
- Each presentation is a VISUAL EXPERIENCE, not an information dump

## 3. Pinho Fornari DNA

### Background
Lead professor of Chora PPT. Background in geology, music industry, and advertising.
Creative director for Loccitane, Nike, Bradesco, Ambev. 15+ years transforming corporate narratives into impactful stories.

### Methodology
- **Narrative architecture BEFORE opening software** — sketch the story first
- **The Villain opens the presentation** — start with tension, not solutions
- **Every slide is a scene** — think like a film director, not a document formatter
- **Contrast is king** — strong visual contrast between ALL elements
- **Alignment and grouping** — clean visual hierarchy through precise placement
- **Images as transformers** — one powerful image > ten bullet points
- **Animations as guides** — movement directs the eye, rhythm controls pacing
- **Typography carries emotion** — font choice is a storytelling decision

### Creative Philosophy
- The best presentations make you FEEL before they make you THINK
- Corporate ≠ boring — sophistication and emotion coexist
- Simplicity is the result of refinement, not laziness
- Every element on the slide must EARN its place

## Unified Style DNA Rules (ALL AGENTS MUST FOLLOW)

### Visual Defaults
| Element | Default | Rationale |
|---------|---------|-----------|
| Background | Dark (black/dark grey) | Premium, sophisticated, high contrast |
| Primary text | White | Maximum readability on dark |
| Accent color | Bold/vibrant (yellow, electric blue, neon) | Energy, disruption, focus |
| Typography | Modern sans-serif, bold weights | Clean, contemporary, impactful |
| Layout | Grid-based, generous whitespace | Breathing room, visual clarity |
| Content density | ONE idea per slide | Radical simplicity |
| Images | Full-bleed, high-resolution | Emotional impact |
| Animations | Purposeful, subtle (0.3s) | Guide attention, never distract |

### Narrative Defaults
| Element | Default | Rationale |
|---------|---------|-----------|
| Opening | Hook in first 90 seconds (Conexao) | Capture attention immediately |
| Structure | Villain → Solution → Moral | Tension-driven narrative |
| Story ratio | 70% story / 30% data | Emotion anchors memory |
| Slide purpose | Support the speaker, not replace | Slides are visual aids |
| Planning ratio | 60% planning / 40% executing | Story before software |

### Anti-Patterns (NEVER DO)
- Wall of text / bullet point lists on slides
- Clip art or generic stock photos
- Default PowerPoint/Google Slides templates
- Reading the slides aloud (slides ≠ teleprompter)
- Starting with the solution (always start with the Villain/problem)
- Decorative animations (spinning, bouncing, flying text)
- Low contrast (grey text on white background)
- Corporate template aesthetic (blue headers, white background, logo on every slide)
- Information dumps (multiple ideas per slide)
- Ignoring the audience's emotional state

### Quality Checkpoint
Before delivering any presentation recommendation, agents MUST verify:
- [ ] Dark mode / high contrast applied?
- [ ] One idea per slide?
- [ ] Narrative structure follows Villain → Solution → Moral?
- [ ] Typography is bold, modern, hierarchical?
- [ ] Images are high-res, purposeful, emotional?
- [ ] Animations guide attention (not distract)?
- [ ] Audience empathy considered in every choice?
- [ ] The presentation makes the audience FEEL something?

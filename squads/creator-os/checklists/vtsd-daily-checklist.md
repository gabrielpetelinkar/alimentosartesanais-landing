# VTSD Daily Execution Checklist

Daily checklist for running your VTSD (Venda Todo Santo Dia) perpetual sales system. Complete every item daily for consistent results.

---

## Morning (30 minutes)

### Metrics Review
- [ ] Check yesterday's CPL (Cost per Lead) — target: R$2-R$15
- [ ] Check landing page conversion rate — target: 25-35%
- [ ] Check sales page conversion rate — target: 1-5%
- [ ] Check checkout conversion rate — target: 30-50%
- [ ] Check ROAS (Return on Ad Spend) — target: 2x minimum
- [ ] Check email open rate — target: 15-25%
- [ ] Check email click rate — target: 2-5%
- [ ] Check total daily revenue and number of sales
- [ ] Check ticket medio (average revenue per customer)
- [ ] Log all numbers in daily tracking spreadsheet

### Diagnosis
- [ ] Identify the weakest funnel stage (lowest vs target)
- [ ] Define ONE action to improve the weakest stage today
- [ ] Check cart recovery emails — any conversions?

---

## Midday (30 minutes)

### Engagement
- [ ] Respond to ALL DMs received (each DM = potential sale)
- [ ] Respond to comments on ads and organic content
- [ ] Remove spam comments from ads
- [ ] Check for new testimonials or results to capture

### Content (if using organic/Stories)
- [ ] Post Stories according to weekly calendar (if using Stories 10x)
- [ ] Publish scheduled content piece (if content marketing day)
- [ ] Share any client results or testimonials

---

## Evening (30 minutes)

### Ad Management
- [ ] Review today's ad performance
- [ ] Pause ads with CTR below 0.5% (after 1,000+ impressions)
- [ ] Scale winning ads by 20-30% if ROAS > target for 3+ consecutive days
- [ ] Check if any new creative tests are needed
- [ ] Verify daily budget was spent (no delivery issues)

### Email
- [ ] Verify tomorrow's scheduled email is ready and correct
- [ ] Check cart recovery sequence is firing properly
- [ ] Review any email bounces or unsubscribes (list hygiene)

### Optimization
- [ ] Log wins and problems in daily journal
- [ ] Note any ideas for A/B tests
- [ ] Plan tomorrow's priority action

---

## Weekly Review (Sunday — 1 hour)

- [ ] Aggregate all daily metrics into weekly summary
- [ ] Calculate weekly ROAS, revenue, profit
- [ ] Identify the #1 bottleneck of the week
- [ ] Plan next week's optimization priority
- [ ] Review A/B test results (any winners to implement?)
- [ ] Clean email list if needed (remove hard bounces)
- [ ] Plan next week's content calendar (if using organic)
- [ ] Adjust ad budgets for the coming week

---

## Monthly Review (Last day of month — 2 hours)

- [ ] Calculate monthly revenue, ROAS, profit
- [ ] Compare to previous month — growth or decline?
- [ ] Identify the 3 biggest wins of the month
- [ ] Identify the 3 biggest problems of the month
- [ ] Set next month's targets (leads, sales, revenue, ROAS)
- [ ] Plan any new traffic sources to test
- [ ] Review upsell/order bump performance — optimize if needed
- [ ] Update sales page with new testimonials
- [ ] Consider scaling budget if ROAS is consistently above target

---

## Quick Diagnostic Guide

**When to change something:**

| Signal | Wait Time | Then Do |
|--------|-----------|---------|
| New ad — no data yet | 3-5 days | Wait for data before judging |
| CPL too high | 7 days of data | Test new creatives and audiences |
| Landing page conv. low | 500+ visitors | Test new headline, simplify page |
| Sales page conv. low | 200+ visitors | Review 12 steps, add social proof |
| Checkout conv. low | 50+ checkout visits | Simplify form, add trust badges, offer PIX |
| Email open rate low | 3+ sends | Test subject lines, check send time |
| ROAS below 1x | 14 days | Fix weakest stage or pause and diagnose |
| ROAS above 3x for 7+ days | Immediately | Scale budget by 20-30% |

**Golden rule:** Erre barato, corrija rapido, reinvista em escala.

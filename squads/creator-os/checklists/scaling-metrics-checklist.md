# Scaling Metrics Checklist

Use this checklist to evaluate whether your perpetual funnel is ready to scale, and to track the key metrics during the scaling process.

---

## Pre-Scaling Readiness Check

Complete ALL items before increasing budget or adding new traffic sources.

### Funnel Health (ALL must pass)
- [ ] Funnel has been running for 30+ days with consistent data
- [ ] ROAS is consistently above 2x for the last 7 days
- [ ] CPL is within target range (R$2-R$15 depending on niche)
- [ ] Landing page conversion rate is above 20%
- [ ] Sales page conversion rate is above 1%
- [ ] Checkout conversion rate is above 25%
- [ ] Cart recovery emails are active and converting

### Revenue Foundation (ALL must pass)
- [ ] Daily revenue is consistent (no wild swings of more than 50%)
- [ ] Ticket medio is calculated and tracked
- [ ] Order bump is live and has a take rate above 10%
- [ ] Upsell is live (even if conversion is low — data is needed)
- [ ] Welcome email sequence is complete (5 emails)
- [ ] Perpetual nurture emails are running (2-3 per week)

### Operations (ALL must pass)
- [ ] Daily metrics are being tracked in a spreadsheet or dashboard
- [ ] There is a process for responding to DMs/emails within 24 hours
- [ ] Cart recovery is automated (not manual)
- [ ] Ad account is in good standing (no policy violations)
- [ ] Payment processing is reliable (no checkout failures)

**READY TO SCALE?**
- All checked → Proceed to scaling
- 1-3 unchecked → Fix before scaling
- 4+ unchecked → NOT ready — optimize the current funnel first

## Scaling Phase 1: Budget Expansion

### Traffic Scaling Rules
- [ ] Increase budget by 20-30% maximum at a time
- [ ] Wait 3-5 days after each increase before evaluating
- [ ] If ROAS drops below 1.5x after increase, pause and diagnose
- [ ] Never scale budget by more than 50% in one step
- [ ] Track ROAS daily during scaling periods

## Red Flags During Scaling

| Red Flag | Action |
|----------|--------|
| ROAS drops below 1.5x for 5+ days | Pause scaling, diagnose weakest stage |
| CPL increases by 50%+ overnight | Check ad fatigue, refresh creatives |
| Sales page conversion drops below 0.5% | Review copy, check page load speed, add proof |
| Checkout conversion drops below 20% | Check for technical issues, simplify form |
| Email open rate drops below 10% | Clean list, test subject lines, check sender reputation |
| Ad account restricted or flagged | Stop scaling, resolve policy issues first |
| Revenue is up but profit is down | Check if costs grew faster than revenue — optimize margins |

# Creator Journey Readiness Checklist

Use this checklist to assess where a creator is and which phase to start at.

---

## Phase 0: Knowledge Organization

- [ ] Creator can articulate their area of expertise
- [ ] Knowledge exists (even if just in their head)
- [ ] At least one domain where they're "10% ahead" of their audience
- [ ] Willing to do a brain dump exercise

**If ALL checked → Skip to Phase 1**
**If ANY unchecked → Start at Phase 0 with @tiago-forte**

---

## Phase 1: Product Creation

- [ ] Knowledge is organized (PARA or similar structure)
- [ ] Top 3-5 teachable topics identified
- [ ] Target student defined
- [ ] Transformation clear (Point A → Point B)
- [ ] At least basic content exists (notes, outlines)

**If ALL checked → Skip to Phase 2**
**If ANY unchecked → Start at Phase 1 with @amy-porterfield**

---

## Phase 2: Offer Design

- [ ] Digital product exists (course, ebook, program)
- [ ] Product validated with at least 1 beta student or pre-sale
- [ ] Curriculum complete with modules and lessons
- [ ] Delivery method established (platform, format)

**If ALL checked → Skip to Phase 3**
**If ANY unchecked → Start at Phase 2 with @russell-brunson**

---

## Phase 3: Launch

- [ ] Value Ladder designed (minimum 3 tiers)
- [ ] Offer stack created (value > 10x price)
- [ ] Epiphany Bridge story written
- [ ] Sales page or sales mechanism ready
- [ ] Unique mechanism/differentiator identified

**If ALL checked → Skip to Phase 4**
**If ANY unchecked → Start at Phase 3 with @jeff-walker**

---

## Phase 4: Audience Building

- [ ] At least 1 successful launch completed
- [ ] Email list of 100+ subscribers
- [ ] Revenue generated from product
- [ ] Post-launch learnings documented

**If ALL checked → Skip to Phase 5**
**If ANY unchecked → Start at Phase 4 with @nicolas-cole**

---

## Phase 5: Perpetual Sales

- [ ] Publishing content consistently (3x/week minimum)
- [ ] Email list growing from content
- [ ] Audience engagement on primary platform
- [ ] Content-to-product pipeline working

**If ALL checked → Skip to Phase 6**
**If ANY unchecked → Start at Phase 5 with @leandro-ladeira**

---

## Phase 6: Recurring Revenue

- [ ] Perpetual funnel live and generating daily sales
- [ ] Revenue not dependent on launches alone
- [ ] Stories/content driving daily conversions
- [ ] Existing customer base to convert to members

**If ALL checked → Ready for Phase 6 with @stu-mclaren**
**If ANY unchecked → Stay at Phase 5 with @leandro-ladeira**

---

## Journey Complete Criteria

- [ ] Organized knowledge base
- [ ] Digital product delivering transformation
- [ ] Irresistible offer with Value Ladder
- [ ] Launch strategy proven and repeatable
- [ ] Content system running consistently
- [ ] Perpetual sales generating daily revenue
- [ ] Membership providing recurring income

**All checked = Creator Journey complete. Focus on optimization and scaling.**

## Course Architecture Notes

- [ ] Module 1 delivers a quick win (student feels momentum early)
- [ ] Each module has exactly ONE clear milestone
- [ ] Each lesson has exactly ONE action item
- [ ] Modules build sequentially (no skipping required)
- [ ] No more than 8 modules (less is more)
- [ ] All fluff removed (only need-to-know, not nice-to-know)
- [ ] Bonuses are separate from core curriculum
- [ ] Lesson format: Hook → Teach → Show → Do → Recap

---

## Lesson Planning Template (Per Lesson)

Use this for each lesson when creating content:

**Lesson [X.X]: [Title]**

| Section | Content |
|---------|---------|
| **Hook** | Why does this matter? What problem does it solve? |
| **Teach** | The concept, framework, or skill |
| **Show** | Example or demonstration |
| **Do** | Action item for the student |
| **Recap** | Key takeaway in one sentence |

**Duration target:** 10-20 minutes per lesson

# FP_H_001 — O Problema Real é o Garanhão Individual

**Type:** Heurística Primária
**Agent:** @frederico-papa
**Evidência:** PMID 16099609 (review de amidas, 2005)

## Enunciado

*"only 30–40% of stallions produce semen suitable for cryopreservation"*

A variabilidade entre garanhões não é ruído estatístico — é o problema central. Um protocolo universal que funciona para 30–40% dos animais é um fracasso de 60–70%.

## Operação

1. Identifique o subgrupo que não responde ao protocolo padrão
2. Investigue a causa específica da falha nesse subgrupo
3. Desenvolva intervenção direcionada ao subgrupo
4. Valide com desfecho clínico (prenhez), não apenas motilidade
5. Publique como protocolo alternativo para esse perfil de garanhão

## Aplicação

- Antes de otimizar um protocolo existente, pergunte: **para quais garanhões isso falha?**
- O "poor freezer" não é exceção — é o próximo objeto de estudo
- A solução para cada subgrupo contribui para o mapa completo da criopreservação equina
